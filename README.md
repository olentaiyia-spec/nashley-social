# Nashley Social Scheduler

A standalone Instagram and Facebook post scheduler prototype. It is intentionally separate from the Campflow folders and runs as plain HTML, CSS, and JavaScript.

## Run locally

From this folder:

```sh
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

The app stores edits in browser local storage, so refreshing the page keeps your draft schedule.
